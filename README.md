# LotusSpider
A scrapy spider

## middleware

### RandomUserAgentMiddleware

配置示例

```
DOWNLOADER_MIDDLEWARES = {
    'scrapy.downloadermiddlewares.useragent.UserAgentMiddleware': None,
    'scrapy_fake_useragent.middleware.RandomUserAgentMiddleware': 400,
}
```

支持的选项

* UER_AGENT_TYPE: static, random, per_user
* UER_AGENT_LIST: ua dict or list

per_user说明

* UER_AGENT_LIST为dict
* dict的key为user_id，值应该定义到request.meta['user_id']下。通常为帐号即可。
* 主要目的是保证user agent与登录帐号是对应的，避免异常登录问题
